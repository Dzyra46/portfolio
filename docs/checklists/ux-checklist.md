# UX Checklist

- [ ] Recruiter understands the developer role within 5 seconds
- [ ] Main CTA is visible in hero section
- [ ] Projects explain problem, solution, stack, and result
- [ ] Skills are grouped clearly by category
- [ ] Experience timeline is easy to scan
- [ ] Contact form is simple (4 fields max)
- [ ] CV download button is easy to find
- [ ] GitHub and LinkedIn links are easy to find
- [ ] Animations enhance content, not distract from it
- [ ] Mobile layout is clean and usable
- [ ] Admin dashboard has clear sidebar navigation
- [ ] Empty states explain what to do next
- [ ] Error states are helpful and actionable
